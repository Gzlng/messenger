# Мессенджер с переводчиком

Мессенджер с переводчиком предназначен для обмена сообщениями в локальной сети и автоматического перевода сообщений с английского языка на русский.

## Установка и запуск

Для начала работы необходимо клонировать репозиторий, перейти в директорию проекта и запустить соответствующие скрипты:

```bash
git clone https://github.com/Gzlng/messenger.git
cd messenger


#Запуск мессенджера (выполняется на всех клиентах):

python messenger.py
